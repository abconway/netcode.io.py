# netcode.io.py
Python 3 implementation of [Netcode.io](netcode.io)


## Dependencies
- Python 3
- Make


## Installation
```
make
```


## Development
```
make develop
```
__Recommended__: Use a Virtualenv
```
virtualenv -p $(which python3) venv
source venv/bin/activate
make develop
```


## Test
```
make test
```
